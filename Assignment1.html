HTML TAGS

HTML tags are like keywords which defines that how web browser will format and display the content.
With the help of tags, a web browser can distinguish between an HTML content and a simple content.
HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

BASIC HTML TAGS

html tag

: All content of your webpage must go inside tags.

head tag

: Contains information about the webpage. : The title tag goes inside head tags. : loading js and css

title tag

: Title of the webpage (what appears in the window/tab of your browser). : The text itself does not appear on webpage.

body tag

: Everything that appears on the webpage should go between these tags

p tag

: Defines a paragraph (text with some space on the bottom and top).
